# Prompt

`npm test` is failing after my latest changes. Please inspect the failure, fix only the related issue, and tell me what verification you ran.

# Expected Skills

- `coding-workflow`
- `task-tracking`
- `git-safety`
- `verification-before-done`

# Should Not Trigger By Default

- `workspace-bootstrap`

# Expected Behavior

Codex should classify this as `bugfix_task`, preserve unrelated user changes, avoid broad refactors, and report the exact test command/result.
